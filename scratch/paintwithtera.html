Paint with Tera

Thanks to CSFirst for the Starter Project

In this project, you will create a painting game using Tera as a stamp.

What you will make

Move the mouse to move Tera around the screen. Use the up and down arrows to make her move faster or slower.

Get Started!

Open the starter project, and press Remix!

Title your project "Paint with Tera"
  • infoHow to add a title
    1. Look at the top of the screen for this text box.



    2. Delete the word "Untitled".

    3. Type in a new name for your project.
Movement

We want Tera to follow the mouse. We will need an event, a loop, and three motion blocks.

Add the blocks to Tera so that when the flag is clicked, forever, Tera will point towards the mouse and move 10 steps, and if on edge bounce.
  • infoHint

Save your project
  • infoHow to save your project
    Click "File" in the top-left corner



    Click "Save Now"



    Your project is now saved!
Coloring

Find the stamp block under Extensions.

Find the change color effect block under looks.

Add these to the forever loop.

Now we need to clear the screen. Add this before the forever loop.

Changing Speed With A Variable

Variables store information for us. Create a new variable under the Variable section. Call it "speed" and define it for all sprites.
  • infoHint
    Like this:

Program the speed variable to increase by 1 when the up key is pressed and decrease by 1 when the down key is pressed

Set the speed to 10 when the green flag is clicked

Add the speed variable to the move 10 steps block.

Set the speed to 10 when the green flag is clicked

Save your project

Congrats! Project complete!
