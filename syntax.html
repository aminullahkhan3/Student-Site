Lesson 1 - Basic Syntax

Background Info

Python is an interpreted language. What does that mean? It means that when you run the code you type, it is interpreted into machine language line by line as it’s executed. Machine language is the language your computer’s CPU speaks. The other main type of language is a compiled language like C++ which is all translated at once before running.

Python is very readable and easy to format. If you’re used to JavaScript or HTML/CSS, you might be expecting some { } or < >. Good news! Python formatting just relies on spacing and a few parentheses ( ) and colons (:).

But why didn’t any of the other stuff in my script pop up?

Any code that starts with a hash (#) is a comment. Comments are bits of code that don’t get executed, so you can use them to make notes, or to explain your code.

# is used for a comment that only takes one line, so if you need more than one line, you’ll need to use ‘’’ or “”” to make a multi-line comment.
