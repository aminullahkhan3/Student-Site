Typing Animation

In this project, you will create a typing animation using keyframes, complete with a flashing cursor.

What you will make

A typing animation. Like this one... |

Get Started!

Create a new folder in your project folder. Name it "Typing Animation".

Create a new html file called "index.html".

Create a new css file called "style.css".

Link the style sheet in the header of your html file.
HTML

Set up the doctype, html, head and body tags in your html page.

Using a <p> tag or any header tag (<h1>, <h2>, <h3> etc.), type the words that you want to be your typing animation.
  • infoExample
    <p>Hello World...</p>
    <h1>My name is...</h1>

Add a typing I-beam within span tags. The I-beam is the same key as backslash (above the return key).
  • infoHint
    <p>Hello World...<span>|</span></p>

Give your paragraph or heading tag an id, and give your span a different id.

CSS

Call the id for your text and set the following properties:

White-space: nowrap will push the text all on one line, so that the paragraph is continuously uncovered in pieces.

Overflow: hidden will hide the words until the animation starts.

Now we will write the animation for the text.

Keyframes are CSS rules that tell our animation what to do.

Use “@keyframes” as a new selector followed by the name of your animation and braces
(make sure your @keyframes is outside of the brackets for the element you’re animating!)

In your CSS file, create a new @keyframes called "type".

Give your keyframe a from{ } and to{ }.
  • infoExample
    @keyframes type {
    from{ }
    to{ }
    }

Tell the keyframe to go from width: 0, to width: 100%.
  • infoExample
    @keyframes type {
    from{width:0; }
    to{width: 100%; }
    }

Now we just need one more animation, for the blinking cursor.

Add a new @keyframes called "blink" to your CSS file.
  • infoExample
    @keyframes blink{
    from{ }
    to { }
    }

We need our I-beam to go from color: transparent; to color: black;
  • infoHint
    @keyframes blink{
    from{color: transparent;}
    to {color: black; }
    }

Now call both animations in the appropriate classes using the animation: property.

If you want the animations to go forever, use the infinite argument in the animation property. Also be sure to give each animation a duration of time.
  • infoHint
    #blinkspan{
    animation: blink 1.5s infinite;
    }

Congrats! Project complete!
