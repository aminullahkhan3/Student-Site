Manipulate Code

In this project, you will manipulate a website's code to change images, background-colors, and text!

What you will make

Experiment with existing HTML and CSS code to make changes to a website. Explore what's behind every webpage.

See the Pen SBCS Code by Camisa (@camisa) on CodePen.

Get Started!

Open the starter project !

Note that if you refresh the codepen site, any changes you made will be lost.
HTML

Header tags (ex. h1, h2, h3, etc.) change text.
Header tags look like this:
<h1>
<h2>

Find a header tag in the HTML tab of the codepen site.

Find the <h1> tag that says “Title of your Website”. Replace "Title of your Website" with a title of your choice.

Keep going to change the other headers in the website

Find the paragraph tags <p>, and change what's inside them to update the information on your website.
CSS

CSS (cascading stylesheet) is used to style your page. You can use CSS to change a website’s background-images, types of fonts, text color, or background-colors.

We’ll be using what’s called inline-CSS to style our websites!

Change the text in the “style” section of any part of your code, customizing your page even further!

Visit colors.commutercreative.com for access to more colors to be able to change!

Change all of the different text colors on your page!

The way that you just changed text-color in the different sections of your page, you can also affect background-color.

Personalize the page by styling the fonts.
Background Images

Change the background picture to an image that you like

First, visit images.google.com and search for your favorite image.

Once you find the the picture on Google, right click on the image and select “Copy image address” from the drop down menu.

Return to your code and paste the copied image address into your code.

Practice by changing other background images on your website.
Maps

Visit google.com/maps and type into the search bar any place in the world that you would like to visit!

Click on the drop down menu.

Select “Share or embed map”.

Click Embed map.

Copy the entire line of code that appears.

Find the iframe tag for the Google Map on your website.

Replace all of the iframe code on your website with the new code that you just copied from Google Maps.

Make any other changes you'd like to the site!

Congrats! Project complete!
