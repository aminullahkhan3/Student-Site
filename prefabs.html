Making Prefabricated Models

So far we have been using Prefabs on the Unity Asset Store.

These are really useful because they have the model we want, as well as the script. From experience, you know that scripting can be time consuming. Making a prefab will save you time and effort because then you don't have to program each individual collectable you put in the game.

Imagine Sonic's rings without prefabs.

Making Copies

We're ready to make copies of our collectable.

To do this, drag it from the Hierarchy into the Project view.

This will make a prefab.

It stores the object and allows us to make copies easily and make changes that affect all of those copies.


Make a folder

Right-click → Create → New Folder

Name it Prefabs

Drag your prefab into there.

Click Original Prefab

Now all you have to do to make copies of it is drag the prefab into the scene.

To organize items in your scene, you can create an empty GameObject to use like a folder.

Create → Create Empty

This is a GameObject with nothing on it besides a transform.

Reset its transform to the default by going to the gear in the transform component and clicking reset

Name it Collectables and drag each collectable onto it to make it a child. Now you can open and close it like a folder by clicking the arrow!
