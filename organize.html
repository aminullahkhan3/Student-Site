Unity Student Guide 3 - Fundamental Project Organization

Creating Folders
Part of building your own world will be finding more assets from the Asset Store

As we continue to download and create new assets, it’s going to be important to organize our project. All of our assets will be in the Assets folder.



Every time we download new assets, there is a new folder made. We want to store these folders somewhere so we can find our assets easily. Right-click on the Assets folder and click Create → Folder. Name it Downloads. Drag any folders from downloaded assets into there. Make a folder for Scenes, too, and drag your scene file into there.

You should now have the following folders:

Downloads

Standard Assets

Scenes

As we move forward, take time every once in a while to organize your files.

This will make your life much easier as you continue to build your game and it gets more complex
