Unity Student Guide: Set Up

Unity Version 2018.3.2f1

Helpful Guides: Unity Manuals

Let's get started!

Open Unity

Click the shortcut on the toolbar or desktop.

Create an account

You’ll need an account to use Unity.

From here on out make sure you are logged in your account in Unity. Multiple students use the same computer, and we don't want accidental deletions, so make sure you're in your account!

Sign in, and take the survey

Be sure to sign out of your Unity account after class each day.

Create a New Project

Choose a 3D project, and name it with your name in the title.

Setting Up Standard Assets

Click Assets

→ Import Package

→ Standard Assets

Note: If this doesn’t show up, you may need to download the Standard Asset package. Ask your instructor for help.

If you need to download the assets, you will also need to import them.

Turn off Auto Light Mapping

This feature causes Unity to run very slowly, and is not super-important for our work, so we will turn it off.

Go to Window

→ Rendering

→ Lighting Settings

Uncheck the Auto Generate box.

Now you're ready to start WorldBuilding. Jump into the next lesson to learn about how to set up your environment, and don't forget to save frequently.
