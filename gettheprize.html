Get the prize!

In this project, you will create a collection game, with an enemy to avoid!

What you will make

Coding Movement

We want Gobo to be controlled with the arrow keys.
Each game starts with an Event.
Events start your code or a set of actions.
Find the event that starts the whole game.

We want to use loops so that if the left arrow key is pressed, then your character points in the left direction and moves 10 steps.
  • help I need a hint
Add an enemy

Find the event that will start the game when the green flag is clicked.

Look in motion to make your enemy forever point towards your main character and move 10 steps.

Make this happen forever.
To do this, we will use a forever loop. Loops repeat code as many times as we want.
A forever loop happens for as long as the game is running.

If the enemy touches the player, the game should end.

See if you can combine an event, forever loop, if then statement, sensing block, and stop all (found in Controls) to make the following sentence true:

When the green flag is clicked, if your enemy is touching your main character then stop all, forever.

Scoreboard

Add a variable for the scoreboard. Call this variable "score".

Go to Variables and click Make a Variable.

Use an event so that when the game starts, the score is set to 0.

If your prize touches your main character, then change the score by 1.

Be sure to make this happen forever!
