Background Info

This lesson will teach you one of the most fundamental parts of programming - boolean logic and decision branching!

Python is a loosely typed language, which means that when you define a variable, you don’t need to tell the interpreter what kind of data is being stored in the variable. The Python interpreter will just figure out what kind of variable each one is when it runs.

Boolean Expressions

Remember when we talked about booleans?

Booleans are either True or False. In programming, we call something a boolean expression, if it evaluates to be either True or False.

Why do we care? Sometimes we want our programs to make a decision based on some information.

There are multiple ways to compare information. We can check equality, using ==, for example:

We can check inequality, using != (not equals).

We can compare values using greater than > , less than < , greater than or equal to >= , and less than or equal to <=.

If I’m writing a voting function, I’ll want a different output based on the user’s age. If the user is under 18, they can’t vote, but if they are 18 or above, they can vote.

If user is 18 or more -> can vote

If user is under 18 -> can’t vote

if, elif and else

Let's write our first if statement.

In this function definition, age is a parameter - a bit of data that the function is expecting to receive when it is called. We will use this age parameter to determine how to respond to the user.

Let’s write our first if statement. Note the syntax - if boolean expression:

If the age given is less than 18, we will tell the user that they are not allowed to vote. This is a boolean expression! Age < 18 will evaluate to either True or False. If it is true, then whatever is indented underneath the if happens next.

What if we try an age over 18?

Nothing happens! We haven’t told our function what to do if the user is over 18. We will do that next.

The second part of an if, else statement is the else.

What do we want to happen if the age is not less than 18? Tell the user that they can vote.

**Note that if and else must be at the same indent level to be paired with each other.**

So if age < 18, the function prints Sorry, you’re too young to vote., and otherwise, it prints Congrats! You can vote.

You try it.

Try and run the function!

What if the user is exactly 18? Let’s give them a special message.

Elif is short for else if. If the age is under 18, the user can’t vote, but if the age is exactly 18, let’s tell them welcome to adulthood, and otherwise just say that they can vote.

The order of if, elif and else matters. If has to be first, and else has to be last.

So if age < 18 -> You can’t vote

Else if age == 18 -> Welcome to adulthood

Else -> You can vote
