The Shape of Color

In this lesson you will:
  1. Get to know the default code
  2. Learn how to color shapes, lines, and the background
  3. Make some mistakes and fix them

Background Info

Processing — the father of p5.js — is a coding environment designed for artists and musicians with little to no experience programming. if you are new to this, we’ll get you making something awesome by the end of this lesson. p5.js is the JavaScript version with the added benefit of running natively on the web. The online editor is easy to use right in the browser with nothing to install.

The first thing that we need to do is log in so our code is saved automagically and can be accessed from anywhere.

Definition Follows Function

At the end of the import, you may see a prompt that reads "API update required". Click on "I made a backup, go ahead!"

Delete the Main Camera from your scene.

Save the scene and your project. You now have a project ready to develop for the HTC Vive!
Testing a game with SteamVR

Open SteamVR from your Desktop.
Be sure that your headset is plugged into the box and that the sensors and controllers are on.

The SteamVR window should show that all components are detected and active.

Press Play on your game.

The first time you run a game you should get this error:
Click "Yes"

You'll then get this dialogue box:
Click "Yes" again.

You'll see the SteamVR Input window open now
Click "Save and generate"

We will use the binding UI later to connect controller actions to actions in our games.

If you haven't saved your changes, you'll now be prompted to. Make sure to click Save.

You can now close the SteamVR Input window and run your game.
Be sure to sign out of Unity when you're done!
